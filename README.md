

# Machine Learning Algorithms From Scratch



This repository contains implementations of core Machine Learning algorithms built entirely from scratch using Python and NumPy, 
without using pre-built model implementations from "scikit-learn".

The focus of this repository is understanding how these algorithms work internally by implementing the complete logic manually.



# Implemented Algorithms


## 1. Linear Regression


Implemented:
- Gradient Descent based Linear Regression
- Ordinary Least Squares (OLS) Linear Regression

Features:
- Manual weight and bias updates
- Cost minimization using gradient descent
- Prediction using learned parameters
- OLS implementation using Normal Equation



## 2. Logistic Regression


Implemented:
- Binary Logistic Regression from scratch

Features:
- Custom sigmoid function
- Manual gradient computation
- Probability prediction
- Binary classification without external ML libraries


## 3. KNN Classifier


Implemented:
- K-Nearest Neighbors Classifier from scratch

Features:
- Euclidean distance calculation
- Neighbor selection logic
- Majority voting classification
- Binary classification examples
- Data visualization using Seaborn



## 4. KNN Regressor


Implemented:
- K-Nearest Neighbors Regressor from scratch

Features:
- Distance-based regression
- Average prediction from nearest neighbors
- Euclidean distance implementation


# Technologies Used
